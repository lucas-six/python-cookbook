# Type Hint for Literal

New in Python *3.8*. See [PEP 586](https://peps.python.org/pep-0586/ "PEP 586 – Literal Types").

## Recipes

```python
from typing import Literal

x: Literal[1, 2, True, False]  # one of 1, 2, True, False
```
