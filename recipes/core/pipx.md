# `pipx` - Install and Run Applications

## Set Up

```bash
pip install pipx

pipx ensurepath
pipx completions
```

## Install packages

```bash
pipx install <pkg> ...
```

## List applications installed

```bash
pipx list
```

## Uninstall applicaitons

```bash
pix uninstall <app> ...
```
